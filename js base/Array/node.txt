> 栈,队列
    -> 栈: 后进先出   push(),pop()
    -> 队列: 先进先出  push(),shift()

> 数组
    ->重排序方法
        -> reverse()    翻转数组
        -> sort()  函数接收两个参数  如果第一个参数应该位于第二个之前则返回一个负数,如果两个参数相等则返回0,如果地体各参数听该位于第二个之后则返回一个正数:
            function(a,b){
                return a-b
            }
    ->操作方法
        -> slice()  返回一个新数组 接收两个参数  起始位置index和结束位置index
        -> concat() 拼接多个数组  (ES6中的... 解构运算)
        -> splice() 替换  接收三个参数  起始位置,  替换项个数, 替换项

    ->遍历方法
        -> every()    数组中每一项都满足的话返回true
        -> some()     只要有一项满足条件 返回true
        -> forEach()  改变数组的中的每一项 没有返回值
                forEach不能改变原数组的每个元素  只能通过数组的index去改变
        -> map()      返回一个新对象
        -> filter()   返回满足条件的所有元素 有返回值
